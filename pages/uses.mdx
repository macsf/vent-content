<!-- wp:heading {"level":3} -->
<h3 class="wp-block-heading">Editor &amp; Terminal</h3>
<!-- /wp:heading -->

<!-- wp:paragraph -->
<p>In the beginning, which was a long time ago, I used <a href="https://www.barebones.com/products/bbedit/" target="_blank" rel="noreferrer noopener">BBEdit</a>. It is a good text editor Later I found <a href="https://www.sublimetext.com/" target="_blank" rel="noreferrer noopener">Sublime Text</a> and I like it a lot for its extensibility. I actually did learn Python in order to write the plugins for ST. Then I heard about <a href="https://atom.io/" target="_blank" rel="noreferrer noopener">Atom</a> and its Javascript-based extension. That was a long time ago.</p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>Right now, I regularly use <a href="https://code.visualstudio.com" target="_blank" rel="noreferrer noopener">Visual Studio Code</a> with the default theme and some plugins (shall list later). When I need to make changes to file quickly, I jump to terminal and use <a href="https://formulae.brew.sh/formula/vim" target="_blank" rel="noreferrer noopener">vim</a>. But when I need to edit code on live production 😱, I use <a href="https://nova.app" target="_blank" rel="noreferrer noopener"><s>Nova</s></a> vim.</p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>I use <a href="https://iterm2.com" target="_blank" rel="noreferrer noopener">iTerm</a> for terminal access with <a href="https://formulae.brew.sh/formula/zsh" target="_blank" rel="noreferrer noopener">zsh</a> as default shell and <a href="https://brew.sh" target="_blank" rel="noreferrer noopener">Homebrew</a> to install some necessary programs. I try to keep things at minimal. </p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>I have been trying <a href="http://warp.dev" data-type="link" data-id="http://warp.dev">Warp</a> on and off but still prefer iTerm.</p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>I use Menlo everywhere that needs monospace but <a href="https://fonts.google.com/specimen/IBM+Plex+Mono">IBM Plex Mono</a> is growing on me. </p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>As of late 2024, <a href="https://developer.apple.com/fonts/">SF Mono</a> is default monospace. Switch to <a href="https://fonts.google.com/specimen/Google+Sans+Code">Google Sans Code</a> in 2026.</p>
<!-- /wp:paragraph -->
